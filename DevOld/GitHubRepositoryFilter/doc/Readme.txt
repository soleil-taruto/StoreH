======================
GitHubRepositoryFilter
======================

----
説明

GitHub-のローカル上のリポジトリをコミット・プッシュしても問題無い状態に保つ。


----
処理内容

・日本語を含むファイル・フォルダ名を US-ASCII に変更する。
・マップデータの圧縮


----
コマンド

GitHubRepositoryFilter.exe GIT-HUB-REPO-UNSAFE-MOD 対象リポジトリ-DIR

	第1引数は安全のため(誤実行防止のため)の固定文字列

	対象リポジトリ-DIR ... 処理対象とするリポジトリまたは、
		処理対象とするリポジトリを直下または配下に持つディレクトリ
